# Components for Nutriso Project
- web components for Nutriso Project.
- Can use storybook to display components and confirm how to use.

## The way of install and start

```
npm install
```
```
npm run storybook
```

## The requirements of environment
- node: ^18.0.0
- vite: ^5.2.0,
- react: ^18.2.0,
- tailwindcss: ^3.4.3,
